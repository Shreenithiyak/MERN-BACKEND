CallBacks

Normal callback
A callback is a function that is passed as an argument to another function and is executed after some operation is completed
callback injection
is a technique where a callback function is injected into another function to customize its behavior.
Array callback
Array callback are the function that we pass as an argument in array method like map,filter,reduce etc...
validation callback
validation callback are the function that we pass as an argument in validation method like some,every etc..

Es6 Features

features & uses

let & const = block scope variable declaration
arrow function = syntatic sugar(single line code) for writing function
template literals = easier way to create string with embedded expressions
destructuring assignment = unpacking values from arrays or properties from objects into Extract values easily.
spread operator = expand iterable into individual elements or merge arrays/objects
rest operator= collect multiple elements into an array or object ,use only in function definition
default parameters = set default values for function parameters
classes = syntactic sugar for creating objects and handling inheritance
modules = organize code into reusable modules with import/export
promises = handle asynchronous operations more effectively
async/await = syntactic sugar for working with promises, making asynchronous code look synchronous
Map & Set = new data structures for storing unique values and key-value pairs
optional chaining = safely access nested object properties without worrying about null or undefined
nullish coalescing operator = provide a default value when dealing with null or undefined
for...of loop = iterate over iterable objects like arrays, strings, maps, sets
generators = create iterators with the ability to pause and resume execution
Symbol = create unique identifiers for object properties
BigInt = handle large integers beyond the safe limit of Number
Proxy = create a proxy for an object to customize its behavior
Reflect = provide methods for interceptable JavaScript operations
